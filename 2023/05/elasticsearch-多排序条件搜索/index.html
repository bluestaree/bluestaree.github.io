Elasticsearch 多排序条件搜索

  2023-05-20   elasticsearch   Elasticsearch  评论 字数统计: 1018字 阅读时长: 3分

需求

​ 根据用户输入的关键词进行短语匹配后，全匹配的数据要排在前面，其次是分词后匹配的数据，然后是部分分词匹配的数据（单个字的分词匹配度 需要低于 词组分词匹配度），最后这些数据要根据人均消费进行排序

创建索引

用到了ik分词器

PUT /local_shop1
{
    "settings": {
        "analysis": {
            "filter": {
                "len": {
                    "type": "length",
                    "min": 2
                }
            },
            "analyzer": {
                "ik_smart_filter_length_less_2": {
                    "tokenizer": "ik_smart",
                    "filter": [
                        "len"
                    ]
                }
            }
        }
    }
,
    "mappings" : {
      "properties" : {
        "cost" : {
          "type" : "double"
        },
        "shopName" : {
          "type" : "text",
          "analyzer" : "ik_max_word"
        }
      }
    }
}

这里使用到了自定义分词器， 在ik_smart的基础上 添加Token Length Filter，主要是为了控制分词最小粒度，其效果如下（模拟用户输错情况）：

{
  "analyzer" : "ik_smart_filter_length_less_2",

查询语句

POST /local_shop1/_doc/_search
{
  "query": {
    "function_score": {
      "query": {
        "bool": {
          "must": [
            {
              "match": {
                "shopName": {
                  "query": "八合里海记速我配牛肉"
                }
              }
            }
          ]
        }
      },
      "functions": [
 
{
          "filter": {
            "match_phrase": {
              "shopName": {
                "query": "八合里海记速我配牛肉"
              }
            }
          },
          "weight": 2
        },
        {
          "filter": {
            "match_phrase": {
              "shopName": {
                "query": "八合里海记速我配牛肉",
                "slop": 4
              }
            }
          },
          "weight": 2
        },
        {
          "filter": {
            "match_phrase": {
              "shopName": {
                "query": "八合里海记速我配牛肉",
                "slop": 10
              }
            }
          },
          "weight": 2
        },
        {
          "filter": {
            "match": {
              "shopName": {
                "query": "八合里海记速我配牛肉",
                "minimum_should_match": "2"
              }
            }
          },
          "weight": 2
        },
        {
          "filter": {
            "match": {
              "shopName": {
                "query": "八合里海记速我配牛肉"
              }
            }
          },
          "weight": 2
        } ,
        {
          "filter": {
            "match": {
              "shopName": {
                "query": "八合里海记速我配牛肉",
                "analyzer" : "ik_smart_filter_length_less_2",
                 "minimum_should_match": "2"
              }
            }
          },
          "weight": 2
        }
      ],
       "boost_mode": "replace"
    }

  },
  "sort": [
    {
      "_score": {
        "order": "desc"
      }
    },
    {
      "cost": { 
        "order": "desc" 
      }
    }
  ]
}

使用functions 结合filter，每满足一个条件都会将结果得分乘2。

这里指定 boost_mode = replace ，最终得分将替换掉 原match计算的得分。 默认为（最终的分 = 原匹配得分 * functions得分)
