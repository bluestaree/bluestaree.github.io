Docker常用命令

  2020-04-18   笔记   docker  评论 字数统计: 334字 阅读时长: 1分

查看当前所有镜像

docker images

查看所有容器

docker ps -a

根据一个镜像创建一个容器并运行 (windows环境)

docker run -d –name mysql_3308 -e MYSQL_ROOT_HOST=% -e MYSQL_ROOT_PASSWORD=tdt123 -p 3307:3306 -v //f/docker/mysql_3308:/var/lib/mysql mysql:5.5

  • –name mysql_3308 … mysql:5.5 设置容器名，及选用镜像（如果没有tag即5.5，则会自动下载最新版的mysql）
  • -e MYSQL_ROOT_HOST=% 允许远程登录
  • -e MYSQL_ROOT_PASSWORD=123456 root登录密码
  • -p 3308:3306 端口映射至宿主机3308
  • -d 后台运行容器，并返回容器ID；
  • -v //f/docker/mysql_3308:/var/lib/mysql 绑定镜像位置到宿主机上
  • –lower_case_table_names=1 不区分大小写

其他栗子

docker run –name mysql -e MYSQL_ROOT_PASSWORD=root -p 3306:3306 -v //e/docker/share/mysql_3306:/var/lib/mysql mysql

docker create –name tracker –net host -v //e/docker/data/tracker:/var/fdfs delron/fastdfs tracker

docker create –name es -p 9200:9200 -p 9300:9300 -e “discovery.type=single-node” -v //e/docker/es/data/:/usr/share/elasticsearch/data elasticsearch:6.5.4

docker create –name kibana -p 5601:5601 -v //e/docker/es/kibana/kibana.yml:/usr/share/kibana/config/kibana.yml kibana:6.5.4

docker create –name redis -p 6379:6379 -v //e/docker/redis/data:/data redis:5.0.2

复制当前目录文件至指定容器位置

docker cp elasticsearch-analysis-ik-6.5.4.zip es:/usr/share/elasticsearch/plugins/

启动容器并显示启动日志

docker start rabbitmq && docker logs -f rabbitmq

进入容器

docker exec -it 775c7c9ee1e1 /bin/bash (进入容器控制台)

  • 775c7c9ee1e1 为容器ID

查看容器信息

dokcer inspect 容器名

查找文件

find / -name tracker.conf

保存镜像文件

docker save

docker save -o python_3.tar python:3

加载镜像文件

docker load

docker load -i python_3.tar
