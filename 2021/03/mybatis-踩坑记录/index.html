Mybatis踩坑记录

  2021-03-23   笔记   mybatis  评论 字数统计: 249字 阅读时长: 1分

最近在自测是发现了不少与mybatis相关错误，虽然解决起来不是很困难，但还是特地记录一下，主要原因还是太粗心了。

问题1：整数类型if语句判断未生效

解决方法：

增加 or soft.rolled==0 或者 删除 soft.rolled!=’’ 条件判断，让mybatis即使数值为0，也进入该判断即可 ,

<if test="user.status != null and user.status != '' or user.status == 0"> and status = #{user.status} <if>

问题2：抛出异常：There is no getter for property named xxx

解决方法：

红框部分内容 应保持一致

另外在这里记录一个可能导致雪花算法生成主键未生效的问题

使用 基本数据类型 作为主键 类型， 新增 情况下 默认 主键为 0 且 无法置为null ，无法使用mybatis雪花算法
